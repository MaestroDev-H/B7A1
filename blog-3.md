# TypeScript Generics: রিইউজেবল এবং টাইপ-সেফ কোড তৈরির টুল

কোড লেখার সময় আমাদের এমন অনেক পরিস্থিতির মুখোমুখি হতে হয় যেখানে লজিক একদম একই থাকে, কিন্তু ডেটার টাইপ ভিন্ন ভিন্ন হতে পারে। এই সমস্যার সমাধানে এবং কোডকে শতভাগ টাইপ-সেফ রেখে রিইউজেবিলিটি বাড়াতে টাইপস্ক্রিপ্টে আবির্ভাব ঘটে **Generics**-এর।

---

## ১. জেনেরিকস আসলে কী?

সহজ কথায়, জেনেরিকস হলো এমন এক ধরণের ভ্যারিয়েবল যা মান বা ভ্যালুর পরিবর্তে **টাইপকে ভ্যারিয়েবল** হিসেবে গ্রহণ করে। অর্থাৎ, আপনি যখন ফাংশন বা ক্লাসটি ব্যবহার করবেন, তখন নির্ধারণ করবেন সেটি কোন টাইপের ডেটা নিয়ে কাজ করবে।

---

## ২. জেনেরিকসের প্রয়োজনীয়তা (একটি বাস্তব উদাহরণ)

ধরা যাক, আমরা এমন একটি ফাংশন তৈরি করতে চাই যা একটি অ্যারের প্রথম উপাদানটি রিটার্ন করবে।

### জেনেরিকস ছাড়া ভুল পদ্ধতি (`any` ব্যবহার করে):

```typescript
function getFirstElement(arr: any[]): any {
  return arr[0];
}

const num = getFirstElement([1, 2, 3]);
```

এখানে আমরা টাইপ সেফটি হারালাম। কম্পাইলার জানবে না যে `num` আসলে একটি নাম্বার।

### জেনেরিকস দিয়ে সঠিক পদ্ধতি:

```typescript
function getFirstElement<T>(arr: T[]): T {
  return arr[0];
}

const num = getFirstElement<number>([1, 2, 3]);
const text = getFirstElement<string>(["a", "b", "c"]);
```

এখানে `<T>` হলো একটি টাইপ প্যারামিটার। যখন আমরা `[1, 2, 3]` পাঠাই, টাইপস্ক্রিপ্ট স্বয়ংক্রিয়ভাবে বুঝে নেয় যে এখানে `T = number`।

## ৩. Generic Constraints: টাইপ সীমাবদ্ধ করা

কখনো কখনো আমরা চাই আমাদের জেনেরিক ফাংশনটি যেকোনো টাইপ গ্রহণ না করে, নির্দিষ্ট একটি শর্ত মেনে চলুক। এর জন্য আমরা `extends` কিওয়ার্ড ব্যবহার করি।

### উদাহরণ:

এমন একটি ফাংশন যা ইনপুট অবজেক্টের `length` প্রপার্টি প্রিন্ট করবে। এর মানে ইনপুট অবজেক্টে অবশ্যই `length` থাকতে হবে।

```typescript
interface HasLength {
  length: number;
}

function logLength<T extends HasLength>(item: T): void {
  console.log(`Length: ${item.length}`);
}

logLength("Hello"); // সফল, স্ট্রিংয়ের length আছে।
logLength([1, 2, 3]); // সফল, অ্যারের length আছে।
// logLength(123); // এরর! নাম্বারের কোনো length প্রপার্টি নেই।
```

## উপসংহার

জেনেরিকস মূলত টাইপস্ক্রিপ্টের আসল শক্তি। এটি ব্যবহার করে আমরা এমন ফ্লেক্সিবল কোড লিখতে পারি যা একই সাথে অত্যন্ত ডাইনামিক এবং শতভাগ টাইপ-সেফ। বড় প্রজেক্টে এপিআই রেসপন্স বা রিইউজেবল কম্পোনেন্ট হ্যান্ডেল করার জন্য জেনেরিকস শেখা আবশ্যিক।
